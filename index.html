Étude linguistique détaillée sur les verbes Tadaksahak :

Some Verb Morphology Features of Tadaksahak

Auteurs : Niels et Regula Christiansen
Publié par : SIL International (2002)

Résumé

Cette étude décrit les caractéristiques morphologiques des verbes en Tadaksahak...

Introduction (Classification et Phonèmes)

Tadaksahak est classé comme langue mixte...

Formes Verbales (Comparatif)

  • Songhay : suffixe causatif typique: -mch
  • Tamasheq : utilisation de préfixes/reflexifs : r-, s-, m-
  • Tadaksahak : préfixes causatifs (sV-), passifs (te-, tuwa-), réflexifs (me-, nem-)

Appendice : Exemples de Verbes

  • adq → causative : rdl˙dq | “être grand”
  • j`x → causative : rdacdc | “s’arrêter”
  • Ó` → causative : ÇhjÇ` | “manger”
